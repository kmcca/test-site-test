Review SBOM with Veracode Container Security

After you run the ./veracode sbom command, Veracode Container Security generates a software bill of materials (SBOM) that appears in your command window or outputs the SBOM to a file.

Include the --format flag to specify the output format of the SBOM, such as CycloneDX, SPDX, or a table. For more details about the supported SBOM formats, see the format flags.

Example Result​

This sample output is the result of an SBOM request in SPDX format.

./veracode sbom --source alpine:latest --type image -f spdx-tag-value
